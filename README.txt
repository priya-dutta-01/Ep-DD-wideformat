1. Install Python 3.9 or higher. Check installation:
python --version

2. Create a Virtual Environment (Recommended) -Open Command Prompt / Terminal in the project folder and run:
python -m venv venv

3. Activate it -
venv\Scripts\activate

4. Install Required Libraries
pip install -r requirements.txt

5. Run -
python export_random_scenarios_rest.py --mapping_file mapping_file.xlsx --workbook_name Book1
